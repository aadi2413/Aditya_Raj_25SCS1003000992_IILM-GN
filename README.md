# 🎬 Mood-Based AI Movie Recommendation System

## 📌 Project Description
This project is a simple AI/ML-based movie recommendation system that suggests movies based on the user's current mood.  
The system takes a short text input from the user, analyzes it using sentiment analysis, and recommends movies accordingly.  
It is designed as a beginner-friendly project to demonstrate basic Machine Learning concepts.

---

## 🚀 Features
- Detects user mood from text input  
- Recommends movies based on detected mood  
- Uses simple and lightweight ML model  
- Beginner-friendly implementation  

---

## 🛠️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Kaggle Datasets  

---

## 📂 Dataset Used
- Movies Metadata Dataset (Kaggle)  
- Twitter Sentiment Analysis Dataset (Kaggle)  

---

## ⚙️ How It Works
1. User enters a sentence describing their mood  
2. Text is processed and cleaned  
3. Sentiment analysis model predicts the mood  
4. Movies are filtered based on mood category  
5. Recommended movies are displayed
